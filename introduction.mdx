## Overview

aisten Documentation provides a centralized hub for your project's docs. You create, organize, and publish documentation effortlessly. This space supports Markdown, MDX components, and custom branding to match your needs.

<Callout kind="info">
  Customize your docs with the brand color `#3B82F6` for consistency across all pages.
</Callout>

## Quick Start

Follow these steps to set up your first documentation page.

<Steps>
  <Step title="Create a Page" icon="plus">
    Navigate to the dashboard and click "New Page". Choose `introduction.mdx` as your template.
  </Step>
  <Step title="Add Content" icon="edit">
    Use H2 headings and components. Start with frontmatter for title and description.
  </Step>
  <Step title="Publish" icon="upload-cloud">
    Save and deploy. Share the live URL with your team.
  </Step>
</Steps>

## Customization Examples

Integrate aisten Documentation into your apps with simple code snippets.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Embed docs widget
  import { AistenDocs } from '@aisten/docs-sdk';

  const docs = new AistenDocs({
    spaceId: 'your-space-id',
    themeColor: '#3B82F6'
  });
  docs.render('#docs-container');
  ```
  ```python
  # Python client example
  from aisten_docs import Client

  client = Client(space_id='your-space-id')
  client.set_theme(color='#3B82F6')
  client.embed(container_id='docs-container')
  ```
</CodeGroup>

## Brand Guidelines

<Expandable title="Detailed Brand Guidelines" default-open="false">

Follow these rules to maintain consistency.

### Primary Color
Use `#3B82F6` for buttons, links, and accents.

```css
:root {
  --brand-primary: #3B82F6;
  --brand-hover: #2563EB;
}
```

### Typography
- Headings: Bold, 1.2em line height
- Body: 16px base, sans-serif

| Element     | Font Size | Weight |
|-------------|-----------|--------|
| H2-H4       | 24px-18px | Bold   |
| Paragraphs  | 16px      | Normal |
| Code        | 14px      | Mono   |

Avoid custom fonts unless specified.

</Expandable>
