## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time collaboration for teams editing docs simultaneously
- Introduced custom themes with brand color support like `#3B82F6`
- New search indexing for faster page lookups

## Improvements

- Enhanced MDX parsing speed by 40% for large documentation sites
- Better mobile responsiveness on documentation previews

## Bug Fixes

- Fixed parsing errors with nested `{` and `}` in code blocks
- Resolved heading hierarchy skips in auto-generated pages

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "improvement"]}>

## New Features

- Integrated Mermaid diagram support via code blocks
- Added `<Update>` component for changelog entries

## Improvements

- Optimized image loading with automatic width/height detection
- Streamlined JSX component escaping rules

## Bug Fixes

- Corrected attribute quoting in `<Card>` and `<Tabs>` components
- Fixed blank line requirements inside JSX containers

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched core MDX documentation generator
- Full component library including `<Steps>`, `<Tabs>`, and `<Columns>`

## Breaking Changes

- Removed H1 headings; use frontmatter titles exclusively
- Mandatory double quotes for all JSX string attributes

## Initial Setup

- Basic YAML frontmatter validation
- Escaped special characters in prose

</Update>

## Upgrade Guide

Follow these steps to update your aisten Documentation site.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run `npm list aisten-docs` in your project root.
  </Step>
  <Step title="Install Latest" icon="download">
    <CodeGroup tabs="npm,yarn">
```bash
npm install aisten-docs@latest
```
```bash
yarn add aisten-docs@latest
```
    </CodeGroup>
  </Step>
  <Step title="Verify Changes" icon="check-circle">
    Review `changelog.mdx` and test parsing with `npm run build`.
  </Step>
</Steps>
